# 🌍 Climate Data Analyst Portfolio

An interactive climate data analysis portfolio built with **Python**, **Pandas**, and **Plotly**.

## 📊 Projects Inside

| # | Project | Description |
|---|---------|-------------|
| 1 | 📈 Global CO₂ Trend | 1900–2024 emissions timeline with key world events |
| 2 | 🏆 Top Emitters | Total vs per-capita comparison across 254 countries |
| 3 | 🌡️ Temperature Impact | Which countries caused the most historical warming |
| 4 | ⚡ Energy Sources | Coal / Oil / Gas breakdown over time |
| 5 | 💰 Emissions vs GDP | Does wealth = more pollution? |

## 🛠️ Tools & Skills
`Python` `Pandas` `Plotly` `Streamlit` `Jupyter Notebook` `Data Visualization` `Climate Analysis`

## 📦 Dataset
**Our World in Data** — CO₂ & Greenhouse Gas Emissions  
274 years of data · 254 countries · 79 variables · [Download here](https://github.com/owid/co2-data)

## ▶️ Run Locally

```bash
# Install dependencies
pip install streamlit plotly pandas

# Run interactive web app
streamlit run app.py

# Or open notebook in VS Code
climate_portfolio.ipynb
```
